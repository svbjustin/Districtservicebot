# District Service Bot

Professioneller Discord-Bot für einen FiveM Script / Development Service ("District Service").

## Features
- Ticket-System (Panels, Kategorien, Transcripts, Logs, Review)
- Ankündigungen
- Giveaways
- Script Shop Übersicht
- Logging
- Utility Commands

## Setup (Kurz)
1. `npm install`
2. `.env.example` zu `.env` kopieren und Werte eintragen
3. `node src/index.js`

Ausführliche Anleitung siehe `docs/SETUP.md`.
